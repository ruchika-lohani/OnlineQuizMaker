A comprehensive web-based quiz application built with Django that allows users to create, manage, and take interactive quizzes.

## Features
- User registration and authentication
- Create quizzes with multiple-choice questions
- Take quizzes with automatic scoring
- Beautiful, responsive UI with gradient design
- Track quiz attempts and scores

## Installation
1. pip install django
2. python manage.py migrate
3. python manage.py runserver

## Access
- Website: http://127.0.0.1:8000
- Admin: http://127.0.0.1:8000/admin

## How to Use
1. Register/Login to your account
2. Create quizzes from "Create Quiz" page
3. Add questions to your quizzes
4. Take quizzes available on home page
5. View your results instantly

## Technologies Used
- Django 4.x
- Bootstrap 5
- SQLite3
- HTML5/CSS3
